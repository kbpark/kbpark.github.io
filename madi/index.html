🎻 신규 곡 등록

ℹ️ 내부 가이드 (v0.6.5)

1. 곡 정보 입력 후 PDF를 업로드하면 에디터 버튼이 활성화됩니다.

2. 에디터 진입 후 악보의 첫 마디를 드래그하면 페이지 전체가 자동 인식됩니다.

3. Undo/Redo 버튼으로 편집 내역을 복구할 수 있으며, JSON Load로 이전 데이터를 불러올 수 있습니다.

4. 편집 완료 시 계산된 마디 수가 자동으로 메인 폼에 입력됩니다.

5. 연습 모드에서는 설정된 마디 수만큼 랜덤하게 출제되며 '상' 평가 시 큐에서 제외됩니다.
